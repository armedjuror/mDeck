How it works

1
Write in Markdown
Use the CodeMirror editor with live preview. Separate slides with ---. Math, code, tables — all supported.
2
Present or export
Launch a full-screen Reveal.js slideshow instantly, or download a PDF with one click.
3
Let AI do it
Connect Claude or any MCP client. Say "make a deck on Newton's laws" — your slides appear automatically.

Slide format

relations-functions.md
## Relations and Functions ### Class 11 · NCERT Chapter 2 Given two non-empty sets $A$ and $B$: $$A \times B = \{(a,b) : a \in A,\ b \in B\}$$ --- ## Key Properties - If $n(A) = p$ and $n(B) = q$, then $n(A \times B) = pq$ - $A \times B \neq B \times A$ in general - $A \times \varnothing = \varnothing$ --- ## Code Example ```python def cartesian_product(A, B): return [(a, b) for a in A for b in B] ```

Let Claude build
your decks

mDeck is an MCP server. Connect Claude.ai with a single URL and it can create, update, and append slides directly — while following your slide format rules automatically.

Works with Claude.ai web, Claude Code, and any MCP-compatible client. Sign in once and Claude remembers your decks, categories, and slide style.

Claude.ai web Claude Code Any MCP client No setup needed
claude code
$ claude mcp add mdeck \ --transport http \ --url https://mdeck.armedjuror.in/api/mcp/ \ --header "Authorization: Bearer YOUR_KEY" ✓ Added MCP server: mdeck > Create a 5-slide deck on Newton's Laws with equations and examples ✓ Created deck: newtons-laws ✓ 5 slides · KaTeX math · code examples
