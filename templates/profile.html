Connect to Claude
Add mDeck as an MCP server so Claude can create and manage your decks directly. Generate an API key above, then add one of the configs below to your Claude client.
Run this command in your terminal (replace YOUR_KEY with your API key): 
claude mcp add mdeck \
  --transport http \
  --url {{ request.scheme }}://{{ request.get_host }}/api/mcp/ \
  --header "Authorization: Bearer YOUR_KEY"
Or add manually to ~/.claude.json: 
{
  "mcpServers": {
    "mdeck": {
      "type": "http",
      "url": "{{ request.scheme }}://{{ request.get_host }}/api/mcp/",
      "headers": {
        "Authorization": "Bearer YOUR_KEY"
      }
    }
  }
}
Add to claude_desktop_config.json (macOS: ~/Library/Application Support/Claude/claude_desktop_config.json): 
{
  "mcpServers": {
    "mdeck": {
      "type": "http",
      "url": "{{ request.scheme }}://{{ request.get_host }}/api/mcp/",
      "headers": {
        "Authorization": "Bearer YOUR_KEY"
      }
    }
  }
}
Restart Claude Desktop after saving the config.
Claude.ai web uses OAuth 2.0. Create an OAuth App above to get a client ID and client secret, then enter them in Claude.ai's MCP settings.
  1. Go to OAuth Apps above and click + New App
  2. Copy the Client ID and Client Secret
  3. In Claude.ai → Settings → Integrations → Add MCP server
  4. Enter the server URL, then the client ID and secret when prompted
